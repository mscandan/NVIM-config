# Welcome to my NeoVim config repository.

- This repository contains my config files for NeoVim.
- Most of the time I am developing TypeScript applications and most common framework that I use is React.

## Installation

- Install NeoVim

  - brew install neovim

- Clone this repository

  ```sh
    git clone https://github.com/mscandan/nvim-config.git
  ```

- Download vim-plug

  ```sh
    curl -fLo ~/.config/nvim/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
  ```

- Open NeoVim and check for health. Run 'checkhealth' on neovim.

- Run plugin installation command in NeoVim. 'PlugInstall'

- Edit keys/mappings.vim file for custom keymappings.
